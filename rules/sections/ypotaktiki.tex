\section*{Υποτακτική}

Mόρια της υποτακτικής:
\begin{itemize}
\item να \ar{ ان }
\item για να \ar{ لِـ }
\end{itemize}


Κατασκευή:
\begin{itemize}
\item Το μόριο της υποτακτικής
\item Το ρήμα στον ενεστώτα
\item Φεύγει το \ar{ ـُ } από την κατάληξη ρήματος και στην θέση του μπαίνει \ar{ ـَ }
\item Όπου υπάχει \ar{ ن } (\ar{ انتِ ، انتُم ، هُم }) φεύγει εντελώς
\item Στα \ar{ انتُم ، هُم } στην θέση του \ar{ ن } μπαίνει ένα \ar{ ا } (δεν προφέρεται)
\end{itemize}
